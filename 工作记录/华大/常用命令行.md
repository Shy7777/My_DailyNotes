## wget下载排除第三方库干扰：
```bash
# 临时重置LD_LIBRARY_PATH为系统默认路径（仅当前终端生效）
export LD_LIBRARY_PATH=/usr/lib64:/usr/lib
# 再次执行wget测试（比如wget --version）
wget --version
```
## wget可断点续传命令：

wget -c 文件.zip -P /home/share/huadjyin/home/wanghaoran/shy/data/prime

## 解压命令：
tar包：  tar -xvf 文件名.tar -C /home/user/test
gz包：  gzip -dk 文件名.gz
zip包：  unzip 文件名.zip -d 目标目录

## 查看当前文件夹文件数量：
```bash
ls -l | grep -v ^total | wc -l
```

## 查看当前文件夹前20个文件
```bash
# 查看当前文件夹下的前20个文件/文件夹
ls | head -200

# 查看指定文件夹（比如 /home/user/documents）下的前20个文件/文件夹
ls /home/user/documents | head -20
```

## 查看当前文件夹随机20个文件
```bash
ls | shuf -n 20
```

## 查看当前文件夹大小
```bash
du -sh
```
