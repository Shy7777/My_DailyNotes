11.盛最多水的容器
思考如何能够达到面积最大，那就是只移动最短边的，原本的思考是移动使得当前面积最大的，这是基于贪心算法的局部最优解，但忽略了全局解，也就是就算当前面积最小，但移动之后仍然可能使得面积最大，只移动最短边可以保证不错过全局最优解。
```python
class Solution:
def maxArea(self, height: List[int]) -> int:
	low, fast = 0, len(height) - 1
	max_area = 0
	while low < fast:
		# 计算当前面积
		current_area = (fast - low) * min(height[low], height[fast])
		max_area = max(max_area, current_area)
		# 移动较短边的指针
		if height[low] < height[fast]:
			low += 1
		else:
			fast -= 1
return max_area
```
