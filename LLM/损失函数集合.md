**快速回答：**损失函数是机器学习和深度学习的核心，用来衡量预测与真实值的差距。不同任务（回归、分类、生成、排序等）有不同的损失函数选择，直接影响模型性能与优化效果。  

---

## 🔑 损失函数分类与应用场景

### 1. 回归任务常用损失函数
- **MSE (Mean Squared Error)**  
  - **公式：** \(\frac{1}{n}\sum (y_i - \hat{y}_i)^2\)  
  - **应用场景：**连续数值预测（房价预测、温度预测）。对大误差敏感。  
- **MAE (Mean Absolute Error)**  
  - **公式：** \(\frac{1}{n}\sum |y_i - \hat{y}_i|\)  
  - **应用场景：**鲁棒性更强，适合含噪声数据。  
- **Huber Loss**  
  - **特点：**结合 MSE 与 MAE，小误差用平方，大误差用绝对值。  
  - **应用场景：**金融预测、医学数据，兼顾稳定性与鲁棒性。  
- **Log-Cosh Loss**  
  - **公式：** \(\sum \log(\cosh(y_i - \hat{y}_i))\)  
  - **应用场景：**平滑近似 MAE，常用于深度回归任务。  
- **Quantile Loss**  
  - **应用场景：**预测分位数（风险管理、天气预报中的上下界预测）。  

---

### 2. 分类任务常用损失函数
- **Binary Cross-Entropy (Log Loss)**  
  - **公式：** \(-\frac{1}{n}\sum [y_i \log \hat{y}_i + (1-y_i)\log(1-\hat{y}_i)]\)  
  - **应用场景：**二分类（癌症检测、垃圾邮件识别）。  
- **Categorical Cross-Entropy**  
  - **应用场景：**多分类任务（图像识别、语音识别）。  
- **Hinge Loss**  
  - **公式：** \(\sum \max(0, 1 - y_i \hat{y}_i)\)  
  - **应用场景：**支持向量机（SVM），强调分类间隔。  
- **Kullback–Leibler (KL) Divergence**  
  - **应用场景：**概率分布匹配（语言模型、生成模型）。  

---

### 3. 生成与概率模型损失
- **GAN Loss (Minimax, Wasserstein)**  
  - **应用场景：**生成对抗网络（图像生成、风格迁移）。  
- **VAE Loss (Reconstruction + KL Divergence)**  
  - **应用场景：**变分自编码器，图像生成、异常检测。  
- **Perceptual Loss**  
  - **应用场景：**图像超分辨率、风格迁移，基于高层特征差异。  

---

### 4. 序列与排序任务损失
- **CTC Loss (Connectionist Temporal Classification)**  
  - **应用场景：**语音识别、手写识别，解决输入输出长度不对齐。  
- **Ranking Loss (Pairwise/Triplet)**  
  - **应用场景：**推荐系统、搜索引擎，优化排序结果。  
- **Contrastive Loss / Triplet Loss**  
  - **应用场景：**度量学习、人脸识别，拉近相似样本、拉远不相似样本。  

---

### 5. 特殊任务损失
- **Dice Loss / IoU Loss**  
  - **应用场景：**医学图像分割，解决类别不平衡。  
- **Focal Loss**  
  - **应用场景：**目标检测（如 RetinaNet），增强对难分类样本的关注。  
- **Custom Multi-task Loss**  
  - **应用场景：**多任务学习，将不同任务损失加权组合。  

---

## 🎯 总结
- **回归 → MSE/MAE/Huber**  
- **分类 → Cross-Entropy/Hinge/Focal**  
- **生成 → GAN/VAE/Perceptual**  
- **序列 → CTC/Ranking/Contrastive**  
- **分割 → Dice/IoU/Focal**  

选择损失函数要结合 **任务类型、数据特性、模型目标**。例如医学病理图像分割常用 **Dice Loss + Cross-Entropy**，而推荐系统更适合 **Ranking Loss**。
